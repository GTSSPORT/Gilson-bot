
GTSSPORT BOT - Instalação Completa

===========================================
=== INSTRUÇÕES PARA LINUX / VPS / UBUNTU ===
===========================================

1. Extraia este ZIP
2. No terminal, vá até a pasta extraída
3. Rode:

chmod +x install_and_run.sh
./install_and_run.sh

Isso irá:
- Instalar dependências
- Instalar o PM2
- Rodar o bot em segundo plano
- Configurar para reiniciar com o sistema

OUTROS COMANDOS:
./stop_bot.sh          -> Para o bot
./restart_bot.sh       -> Reinicia o bot
./update_bot.sh        -> Atualiza o código do bot (index.js)

=======================================
=== INSTRUÇÕES PARA WINDOWS (CMD) =====
=======================================

1. Extraia o ZIP
2. Clique duas vezes no arquivo:
   start_bot_windows.bat

Isso irá:
- Instalar dependências
- Instalar PM2
- Rodar o bot com PM2
- Configurar para iniciar automaticamente

========================================
== SOBRE O BOT ==
- Funciona com o WhatsApp via Venom Bot
- Usa OpenAI com chave já embutida
- Mostra QR Code no início
- Responde com:
  • Link do vídeo de demonstração
  • Link de pagamento
  • Link para download
  • Mensagem carinhosa para pessoa especial
- Funciona mesmo se o terminal for fechado!
